# Slime-Mold-Training
Slime molds are undeniably awesome! They look alien, but also posess rudimentary intelligence and are able to solve mazes and optimize systems without a brain. This project aims to use selective breeding to engineer a slime mold to run faster, as most slime molds can run about 5mm per hour. To achive this goal, I will use Physarum polycephalum, a fairly common slime mold as my subject of interest. I chose Physarum polycephalum because it is easy to feed and propigates easily. To selectively engineer this slime mold, I will setup a tank of multiple slime molds growing on damp paper. I will place a oat at the end of a strip of damp paper, and whichever slime mold gets there first, I will propigate that slime mold and repeat the process untill I have selectively bred the fastest slime mold.
